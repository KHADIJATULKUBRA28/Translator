# the three commands you need to know to use git

- `git add .`: This command stages the changes you have made to a file. It tells git that you want to include these changes in the next commit.

- `git commit -m "Your commit message"`: This command creates a new commit with the changes you have staged. The `-m` flag allows you to include a message describing the changes you made.

- `git push`: This command uploads your local commits to a remote repository, such as GitHub. It allows others to see your changes and collaborate with you on the project.
